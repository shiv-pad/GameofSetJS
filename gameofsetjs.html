Game of Sets

The objective of set is to find a set of 3 cards from set of cards laid out on your screen.

A deck consists of 81 unique cards which contain the following four attributes:

  • Shape: Oval, Squiggle, Diamond
  • Color: Red, Green, Blue
  • Number of Shapes: One, Two, Three
  • Shading: Solid, Stripped, Open


Rules of Set

A set consists of 3 cards where with each attribute:

  • Each card share the same attribute
  • Each card has a completely different attribute
